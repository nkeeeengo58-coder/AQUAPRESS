# AI module plan

This folder is reserved for future AI-assisted script generation.

Planned direction:
- Use Gemini Flash for low-cost script generation and summaries.
- Prefer free tiers and low-cost APIs first.
- Keep prompts and output schemas small and stable.
- Consider OpenAI API only after monetization or when higher quality is required.

Expected future responsibilities:
- Convert fetched aquarium news into short-form scripts.
- Produce hook, narration, captions, and CTA text.
- Support multiple content genres with the same schema.
